# blucli

A convinient CLI wrapper over bluetoothctl for managing my bluetooth devices.

## Installing

### Requirements
1. bluetoothctl
2. cargo

### Setup
Fetch the sources from github
> `git clone https://github.com/Brannigan123/blucli.git`

Switch into the source directory
> `cd blucli` 

Install the module
> cargo install --path .

## Running it
> `blucli`
